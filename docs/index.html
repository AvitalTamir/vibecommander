Features

Everything you need, nothing you don't

Built from scratch for AI-assisted workflows. No plugins required.

Instant startup

Single binary, no dependencies. Launches faster than your IDE can even think about it.

Dual-pane layout

File tree on the left, code viewer on the right. Resize with a keystroke. Go fullscreen when you need focus.

AI-first design

Git integration

See file status at a glance. Stage, unstage, and commit without leaving the app. Inline diff viewer included.

Fast search

Fuzzy file filtering. Regex search in the code viewer. Find what you need without context switching.
